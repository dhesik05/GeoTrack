# GeoTrack - Geographic Location Tracking System

GeoTrack is a simple location tracking system that retrieves the user's geographic coordinates and displays the location on a map.

## Features
- Detects user latitude and longitude
- Displays location on Google Maps
- Web interface using Flask

## Technologies
Python
Flask
Geolocation API

## Usage

Run the application:

python app.py

Open browser:

http://127.0.0.1:5000
